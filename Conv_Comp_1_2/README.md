#### Instrucciones:

Realiza un programa que calcule el Complemento a 1 y el Complemento a 2

#### Condiciones:

- Solicitar un número decimal positivo.
- El número debe estar dentro del rango macimo de 8 bits.
- Si el número sobrepasa el rango permitido, mostrar un mensaje de error de rango.
- Mostrar la conversión del número decimal a binario (8 bits).
- Calcular el Complemento a 1 y mostrar el resultado.
- Calcular el Complemento a 2 y mostrar el resultado.
- Mostrar la comprobación del resultado obtenido.
